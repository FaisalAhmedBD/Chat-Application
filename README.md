# Event-Calender
### Used Technologies
1. Backend
   - Express.js, Socket.io
2. Frontend
   - Reactjs, bootstrap
##  Features
      - Send message
      - Getting notification when others are typing
 
# Build instruction
1.  clone the repo
2.  move to Backend and Frontend sub-folder and run `npm install`
